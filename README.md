# MiYolla
All-in-one tool for Huami wearables

## Description
* MiYolla is a content aggregator for Huami wearables
* There are 3 categories available
  1. Firmwares feed
  2. Wearable's dials
  3. Android apps catalog
 
## Supported languages
* English
* Russian
* Belarusian
* Ukrainian
* Turkish
* Spanish
* German
* French
* Chinese
* Korean
* Japanese
* Hindi

## Supported platforms
1. Android 4.1 or above
2. Windows 10 or above
